# Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. This project was created to strengthen my understanding of JavaScript fundamentals, DOM manipulation, and event handling through hands-on practice.

## Live Demo

https://jatin-jay-31.github.io/Rock-paper-scissor-game/

## Features

* Classic Rock, Paper, Scissors gameplay
* Play against the computer
* Live score tracking for both player and computer
* 10-round game system
* Round-by-round result display
* Final winner announcement after 10 rounds
* Reset button to start a new game instantly
* Dynamic UI updates with smooth message transitions

## What I Practiced

* DOM selection and manipulation
* Event listeners
* Conditional logic
* Functions and code organization
* Updating the UI dynamically based on game state
* Managing game state with variables
* Using `setTimeout()` for temporary messages

## Challenges I Faced

The most challenging part of this project was implementing the game logic for a fixed 10-round system. It took careful thinking to:

* Track the number of rounds correctly
* Stop the game after 10 rounds
* Prevent further clicks once the game ends
* Determine the final winner based on scores
* Handle draw situations properly, both for individual rounds and for the final result

Another tricky part was managing temporary UI messages, such as displaying the round winner and automatically hiding it after a short delay.

## What I Learned

This project helped me understand how JavaScript brings interactivity to web pages. More importantly, it taught me how to break a problem into smaller parts, build logic step by step, and debug issues along the way.
